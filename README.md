# Task Board Course

A Trello-lite personal task board, built module by module to learn:

- Git + GitHub workflow
- API-first design with OpenAPI
- Node.js backend
- Postgres (SQL) + a NoSQL companion store
- Next.js frontend
- Docker + CI/CD
